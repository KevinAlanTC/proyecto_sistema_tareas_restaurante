# Paquete utils

## Descripción
Funciones auxiliares y utilitarias para apoyar la lógica del sistema.

## Contenido
- `Validador.java`: validaciones de datos (emails, roles, etc.).
- `FechaHelper.java`: manejo de fechas y vencimientos.

## Responsabilidad
Proveer herramientas reutilizables que no pertenecen directamente a los modelos ni a la lógica de negocio.

## Notas
- Mantener métodos estáticos para facilitar su uso.
- Evitar dependencias circulares con otros paquetes.
